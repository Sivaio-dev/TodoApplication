# Todo Application 📝

A full-stack Todo web application built using **Spring Boot** and **PostgreSQL** for backend, with a HTML, CSS and JavaScript for frontend.
The application includes user authentication (login/register) and allows users to manage their personal todos.

---

## 🚀 Features

- User registration and login
- Add new todos
- View all todos
- Mark todos as completed
- Delete todos
- Persistent storage using PostgreSQL

---

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
